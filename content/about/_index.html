Сайт посвященный интеграционным процессам в D365FO (как правильно готовить интеграцию)

{{< /blocks/cover >}} {{% blocks/lead %}} Зачем, что делает, как пользоваться Что делает сайт Сайт предназначен для поиска информации о возможных интеграциях. {{% /blocks/lead %}} {{< blocks/section >}}

В задачи сайта входят размещение информации о известных методах интеграций, ограничениях этих интеграций, плюсах и минусах. Коме этого, для зарегистрированных пользователей возможно подгрузить свою структуры энтити. Обозначить энтити, которые участвуют в интеграционных процессах, подписаться на получение новостной рассылки по изменениям в энтити, которые интересуют зарегистрированного пользователя.

{{< /blocks/section >}} {{< blocks/section >}}

С каждой новой версией от Майкрософт, информация на сайте будет обновляться. Как им пользоваться Два режима. Обычный пользователь - навигация, поиск, комментарии. Зарегистрированный пользователь - все что и обычный + подписка для энтити, возможность сравнения версий по интересующим энтити.
